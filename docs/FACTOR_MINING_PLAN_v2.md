> 基于数据修复后的经验教训，从头设计严谨的因子挖掘流程

---

## 1. 背景与教训

### 1.1 之前的问题

| 问题 | 影响 | 教训 |
|------|------|------|
| 初始数据字段顺序错误 | 所有指标计算错误 | 上游数据必须验证 |
| 因子IC基于错误数据 | 权重无参考价值 | 数据质量是一切基础 |
| 无IC验证流程 | 无法量化因子有效性 | 必须有验证机制 |
| 固定权重无依据 | 主观判断成分大 | 需数据驱动 |

### 1.2 新方案原则

1. **数据质量优先**：每步输入都做验证
2. **可回溯验证**：所有结论需有数据支撑
3. **渐进式挖掘**：小步验证，快速迭代
4. **留痕归档**：每步决策记录在案

### Step 3: IC计算

**目标**：量化每个因子的预测能力

**计算方法**：
```python
# IC = Correlation(因子值, 未来收益)
# IR = IC_mean / IC_std

IC > 0: 因子值越大，收益越高
IC < 0: 因子值越大，收益越低
|IC| > 0.05: 因子有预测能力
IR > 0.5: 因子稳定有效
```

**验证维度**：
| 指标 | 计算方法 | 判定标准 |
|------|----------|----------|
| IC均值 | mean(rolling_IC_20) | > 0.02 有效 |
| IC标准差 | std(rolling_IC_20) | 越小越稳定 |
| IR | IC均值 / IC标准差 | > 0.5 稳定 |
| IC衰减 | 不同持有期IC对比 | 确认有效期 |

**交付物**：
```
factor_ic_report.md
├── 各因子IC/IR汇总表
├── IC时序图（判断稳定性）
├── IC衰减曲线（确认有效期）
└── 有效因子清单
```

**通过标准**：
- IC均值绝对值 > 0.02
- IR > 0.3（放宽标准，适应市场变化）
- 统计显著（样本量 > 100）

### Step 3.5: 第一性原理 - 因子去重分析

**原理**：量化因子存在三类重复，应在计算IC前剔除

| 重复类型 | 定义 | 处理方式 |
|----------|------|----------|
| **信息重复** | 多个因子反映同一信息 | 合并/保留信息量最大的 |
| **数学包含** | A因子由B因子计算而来 | 只保留基础因子 |
| **逻辑等价** | 触发条件高度相关 | 保留更通用的 |

**19个因子的去重分析**：

| 因子A | 因子B | 重复类型 | 去重建议 |
|-------|-------|----------|----------|
| MA5 | MA20/34 | 数学包含 | 合并为MA5+MA34交叉 |
| MA5向上 | MA5 | 逻辑包含 | 保留基础MA5 |
| MAOBV | OBV | 数学包含 | 只保留OBV |
| RSI | WR/CCI | 数学相关 | 只保留RSI |
| MACD | KDJ | 信息重叠 | 互补保留两者 |
| 布林带 | 波动率 | 数学包含 | 布林带已含波动率 |

**合并后核心因子（建议9个）**：

| # | 核心因子 | 合并来源 | 去重理由 |
|---|----------|----------|----------|
| 1 | **MA5>MA34** | MA5/34/向上/MA20 | 均线交叉信号 |
| 2 | **SAR** | - | 独立止损信号 |
| 3 | **MACD** | - | 核心动量 |
| 4 | **KDJ** | - | 互补动量（灵敏）|
| 5 | **OBV>MAOBV** | OBV/MAOBV(10) | 量能资金流入 |
| 6 | **布林带** | 波动率 | 已含波动 |
| 7 | **RSI(5/10)** | RSI(5)/RSI(10) | 短期/中期超买超卖 |
| 8 | **DMA** | 动量 | 趋势差值 |
| 9 | **ADX** | VHF | 趋势强度 |

**结论**：19 → 8，精简核心，消除冗余

### Step 4: 方向判断

**目标**：确定因子使用方向

**决策规则**：
```
IC > 0:
  └── 因子值越大越好 → 保持原方向

IC < 0:
  ├── 理论支持反序 → 反转使用（如RSI越低越买入）
  └── 理论不支持   → 舍弃该因子
```

**示例**：
| 因子 | IC值 | 方向判断 | 用法 |
|------|------|----------|------|
| MA60站上 | +0.05 | ✅ 正向 | 站上MA60买 |
| RSI低位 | -0.08 | ✅ 反转 | RSI<30买 |
| 波动率高 | -0.02 | ❌ 舍弃 | - |

### Step 5: 权重确定

**目标**：根据IC值分配因子权重

**权重公式**：
```python
# 方案A: IC绝对值比例
weight_i = |IC_i| / sum(|IC|) * 总权重

# 方案B: IC绝对值归一化
weight_i = |IC_i| ** alpha / sum(|IC|^alpha)
# alpha=1 时退化为方案A
# alpha>1 时放大高IC因子权重
```

**权重约束**：
- 单因子权重上限：30%
- 最低权重：5%
- 总权重：100%（或等效分数）

**交付物**：
```

├── 权重计算过程
├── 最终权重表
└── 与旧权重对比
```
