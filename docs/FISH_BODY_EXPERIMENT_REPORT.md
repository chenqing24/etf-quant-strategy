## 三、深度分析

### 3.1 关键发现

#### 发现1: Exp1表现异常

```
配置: stop_loss=-0.05, stop_profit=0.08, max_hold_days=15
结果: 收益44.1%, 夏普18.56, 回撤0.0%, 交易6次

⚠️ 异常点:
- 回撤0.0% 不正常
- 交易只有6次
- 夏普18.56 极高

可能原因:
1. 测试数据时间段特殊(全是上涨)
2. 参数过于保守，错过了很多交易
3. 交易信号少但精准
```

#### 发现2: 参数敏感度

| 参数 | 变化范围 | 对收益影响 |
|------|----------|------------|
| 止损 | -2% ~ -10% | 高敏感 |
| 止盈 | 4% ~ 15% | 中敏感 |
| 持仓天数 | 5 ~ 25天 | 低敏感 |

#### 发现3: 鱼身交易占比

```
最佳配置: 鱼身交易占比 92.9%
说明: 鱼身过滤器有效，能避开鱼尾
```

### 3.2 问题诊断

```
【问题1】夏普过高(18.56)可能导致过拟合
- 正常夏普在0.5-2.0之间
- 18.56说明策略太特殊

【问题2】回撤为0不符合实际
- 极端保守的参数导致几乎不交易
- 没有真正测试风险承受能力

【问题3】交易次数偏少
- 最少只有5次，最多11次
- 样本量不足以验证策略稳定性
```
