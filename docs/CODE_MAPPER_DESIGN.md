# ETF代码适配系统设计文档

> 统一管理各数据源ETF代码格式差异

---

## 1. 背景与问题

### 1.1 问题描述

同一个ETF在不同数据源的代码格式不同：

| 数据源 | 深交所ETF | 上交所ETF |
|--------|----------|----------|
| **内部标准** | 159806 | 510300 |
| **腾讯** | 159806 | 510300 |
| **BaoStock** | sz.159806 | sh.510300 |
| **东方财富** | sz159806 | sh510300 |
| **新浪** | sz159806 | sh510300 |
| **同花顺** | 159806 | 510300 |

### 1.2 影响

- 代码散落在各模块，维护困难
- 新增数据源需要重复适配
- 容易出错，难以调试

---

## 2. 解决方案：方案C - Router层统一转换

### 2.1 架构设计

```
┌─────────────────────────────────────────────────────────┐
│                    Router 层                            │
│  ┌─────────────────────────────────────────────────┐    │
│  │  ETFCodeMapper.adapt(code, source)              │    │
│  │  标准码 → 目标数据源格式                          │    │
│  └─────────────────────────────────────────────────┘    │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│                    适配器层                             │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐   │
│  │ Tencent │ │BaoStock │ │EastMoney │ │  Sina   │   │
│  │  Adapter│ │  Adapter│ │  Adapter│ │  Adapter│   │
│  └──────────┘ └──────────┘ └──────────┘ └──────────┘   │
└─────────────────────────────────────────────────────────┘
```

### 2.2 代码映射规则

| 数据源 | 深交所格式 | 上交所格式 |
|--------|-----------|-----------|
| **标准** | 159806 | 510300 |
| **腾讯** | 159806 | 510300 |
| **BaoStock** | sz.159806 | sh.510300 |
| **东方财富** | sz159806 | sh510300 |
| **新浪** | sz159806 | sh510300 |

### 2.3 市场判断规则

```python
# 上交所: 510xxx, 511xxx, 512xxx, 513xxx, 515xxx, 516xxx, 518xxx
# 深交所: 159xxx, 161xxx 等

PREFIX_SH = ('510', '511', '512', '513', '515', '516', '518')
PREFIX_SZ = ('159', '161')
```

---

## 3. 接口设计

### 3.1 ETFCodeMapper 类

```python
class ETFCodeMapper:
    """ETF代码映射器 - 统一转换各数据源格式"""
    
    # 市场前缀定义
    PREFIX_SH = ('510', '511', '512', '513', '515', '516', '518')
    PREFIX_SZ = ('159', '161')
    
    @staticmethod
    def to_standard(code: str) -> str:
        """任意格式 → 标准6位码
        
        Args:
            code: 任意格式的ETF代码
            
        Returns:
            标准6位数字码
            
        Examples:
            'sz.159806' → '159806'
            'sh510300' → '510300'
            '159806' → '159806'
        """
        
    @staticmethod
    def to_source(code: str, source: str) -> str:
        """标准码 → 目标数据源格式
        
        Args:
            code: 标准6位ETF代码
            source: 数据源名称 (tencent/baostock/eastmoney/sina)
            
        Returns:
            目标数据源格式的代码
        """
        
    @staticmethod
    def get_market(code: str) -> str:
        """判断市场
        
        Args:
            code: 标准6位ETF代码
            
        Returns:
            'SH' (上海) 或 'SZ' (深圳)
        """
```

### 3.2 支持的数据源

| 数据源标识 | 说明 |
|------------|------|
| `tencent` | 腾讯财经API |
| `baostock` | BaoStock |
| `eastmoney` | 东方财富 |
| `sina` | 新浪财经 |
| `standard` | 内部标准格式 |
