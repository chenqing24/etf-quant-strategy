# 交易成本规范

> 规范ETF交易的成本计算模型

## 1. 概述

交易成本模块计算ETF买卖过程中的各项费用，包括佣金、印花税（仅卖出收取）、过户费等。

## 2. 费用构成

### 2.1 费用类型

| 费用类型 | 收取方式 | 标准费率 |
|----------|----------|----------|
| 佣金 | 双向收取 | ≤0.03% (最低5元) |
| 印花税 | 仅卖出 | 0.05% |
| 过户费 | 双向收取 | 0.002% |
| 管理费 | 无 | 0 |
| 托管费 | 无 | 0 |

### 2.2 ETF vs 股票

```
ETF优势：免收印花税！
```

| 品种 | 买入成本 | 卖出成本 |
|------|----------|----------|
| 股票 | 佣金+印花税 | 佣金+印花税+过户费 |
| ETF | 佣金 | 佣金+过户费 |

## 3. 成本计算

### 3.1 Python实现

```python
class TradingCostCalculator:
    """交易成本计算器"""
    
    COMMISSION_RATE = 0.0003  # 0.03%
    STAMP_TAX_RATE = 0.0005  # 0.05% (仅卖出股票)
    TRANSFER_FEE_RATE = 0.00002  # 0.002%
    MIN_COMMISSION = 5.0  # 最低佣金
```

### 3.2 计算公式

**买入成本**：
```
买入佣金 = max(成交金额 × 佣金率, 最低佣金)
买入费用 = 买入佣金
```

**卖出成本**：
```
卖出佣金 = max(成交金额 × 佣金率, 最低佣金)
印花税 = 成交金额 × 印花税率
过户费 = 成交金额 × 过户费率
卖出费用 = 卖出佣金 + 印花税 + 过户费
```

**总成本**：
```
总成本 = 买入费用 + 卖出费用
单边成本率 = 总成本 / 成交金额 / 2
```

## 4. 实际案例

### 4.1 买入示例

```
买入: 515050 @ 1.101元 × 10000份 = 11010元
佣金: max(11010 × 0.0003, 5) = 5元 (最低)
费用: 5元
费率: 5 / 11010 = 0.045%
```

### 4.2 卖出示例

```
卖出: 515050 @ 1.189元 × 10000份 = 11890元
佣金: max(11890 × 0.0003, 5) = 5元
印花税: 11890 × 0 = 0元 (ETF免收)
过户费: 11890 × 0.00002 = 0.24元
费用: 5.24元
费率: 5.24 / 11890 = 0.044%
```

## 5. 盈亏平衡计算

### 5.1 公式

```
盈亏平衡涨幅 = (买入费率 + 卖出费率) × 2
```

### 5.2 示例

```
单边费率 ≈ 0.045%
盈亏平衡涨幅 ≈ 0.09%
```

即：**涨幅超过0.09%即可覆盖交易成本**

## 6. 使用示例

```python
from src.trading_cost import TradingCostCalculator

calculator = TradingCostCalculator()

# 计算买入成本
buy_cost = calculator.calculate_buy_cost(
    price=1.101,
    quantity=10000
)

# 计算卖出成本
sell_cost = calculator.calculate_sell_cost(
    price=1.189,
    quantity=10000
)

print(f"买入费用: {buy_cost:.2f}元")
print(f"卖出费用: {sell_cost:.2f}元")
print(f"总成本: {buy_cost + sell_cost:.2f}元")
```
