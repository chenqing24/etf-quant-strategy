# ETF量化策略 - 输出格式规范

## 1. 场景定义

| 场景 | 渠道 | 输出内容 | 格式 |
|------|------|----------|------|
| **mobile** | 钉钉APP | 精简结论 | Markdown，约10行 |
| **pc** | PC网页端 | 完整报告 | 详细文本+文件链接 |
| **console** | 命令行 | 正常输出 | 带进度条+详细日志 |

## 2. 各场景输出规范

### 2.1 Mobile（钉钉APP）

**精简版Markdown格式**：

```markdown
## 📈 ETF量化决策 05-26 13:04

**🟢 买入** 515050 科技50
信号价: 1.101 | 实时: 1.196 (+8.6%)  
止损: 1.046 | 止盈: 1.189
---
历史: +91.6% | 回撤: -6.7% | 夏普: 1.31
```

**特点**：
- 约10行内
- 只包含：操作、价格、止损止盈、历史指标
- 行尾加2个空格（钉钉换行要求）
- 无进度条、无中间过程

### 2.2 PC（网页端）

**详细报告格式**：

```markdown
======================================================================
📈 ETF量化投资决策报告
======================================================================

【基本信息】
报告生成时间: 2026-05-26 13:04
数据最新日期: 2026-05-22
投资本金: 20,000元

======================================================================
🚨 今日交易建议
======================================================================

【操作】买入 515050 科技50
【价格】1.101元
【数量】15000股

... (完整报告内容)
```

**特点**：
- 完整指标展示
- 包含：市场分析、策略历史、多时段验证
- 详细RSI、趋势图数据

### 2.3 Console（命令行）

**带进度条的标准输出**：

```
============================================================
🔄 完整策略评估
============================================================

[预热] 拉取实时数据...
✅ 25只ETF预热完成

[1/3] 生成决策报告...
✅ 报告已保存: etf_reports/report_20260526.txt

[2/3] 分析建议...
  今日操作: 买入 515050

[3/3] 发送通知...
  ✅ 钉钉推送成功
```

**特点**：
- 显示进度和步骤
- 中间过程日志
- 适合开发调试

## 3. 实现架构

```
decision_cli.py
    │
    ├── --mode eval
    │       ├── scenario='console' → print_full_report()
    │       ├── scenario='mobile'  → build_simple() → 精简输出
    │       └── scenario='pc'      → build_full() + 文件链接
    │
    └── --mode daily
            └── scenario='mobile' → 自动精简模式
```

## 4. 调用方式

### 命令行

```bash
# 默认（控制台模式）
python -m src.decision_cli -m eval

# 简版输出（钉钉场景）
python -m src.decision_cli -m eval --simple

# 完整报告（PC场景）
python -m src.decision_cli -m eval --full
```

### Python调用

```python
from src.scenario_adapter import ScenarioAdapter

# 钉钉场景
adapter = ScenarioAdapter.for_mobile()
message = adapter.build_report(results)
```
