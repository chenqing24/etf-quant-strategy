## 3. 不纳入的ETF

以下ETF不纳入本系统管理：

| 类型 | 原因 |
|------|------|
| **跨境海外ETF** | 海外API（如纳指ETF 513100）不纳入 |
| **规模过小** | 规模<10亿、成交<1000万的ETF |
| **流动性差** | 日均成交<100万的ETF |
| **已退市** | 已在交易所退市的ETF |
