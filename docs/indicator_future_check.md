# 技术指标未来函数深度分析

## 一、核心指标代码审查

### 1.1 ADX (Average Directional Index)

```python
# ADX计算源码分析
high_close = (high - close.shift(1)).abs()  # ✅ 使用shift(+1)，引用前日数据
low_close = (low - close.shift(1)).abs()    # ✅ 使用shift(+1)，引用前日数据
```

**结论**: ✅ **无未来函数**
- TR计算使用 `shift(1)` 引用前日收盘价
- 使用 `rolling().sum()` 平滑，只用历史数据
- ADX本身是滞后指标，无look-ahead bias

---

### 1.2 Bollinger Bands (布林带)

```python
# BB计算源码分析
middle = close.rolling(window=20, min_periods=1).mean()  # ✅ 只用历史均线
std = close.rolling(window=20, min_periods=1).std()      # ✅ 只用历史标准差
upper = middle + 2 * std                                 # ✅ 基于历史数据
lower = middle - 2 * std                                 # ✅ 基于历史数据
percent = (close - lower) / (upper - lower) * 100       # ✅ 使用当日数据
```

**结论**: ✅ **无未来函数**
- 均线和标准差都是滞后计算
- 布林带位置使用当日价格和历史统计量
- 无shift(-1)引用

---

### 1.3 SAR (抛物线止损)

```python
# SAR递归计算分析
for i in range(1, n):
    current_sar = prev_sar + prev_af * (prev_ep - prev_sar)  # ✅ 只用前日数据
    # 检查价格穿越
    if low.iloc[i] < current_sar:    # ✅ 使用当日价格
        current_sar = prev_ep        # ✅ 使用前日极点价
```

**结论**: ✅ **无未来函数**
- SAR是递归计算，每步只用前日SAR和当日价格
- 趋势反转判断基于当日高低价
- 无未来数据依赖

---

### 1.4 OBV (能量潮)

```python
# OBV计算源码分析
price_change = close.diff()              # ✅ 比较相邻日价格变化
obv[price_change > 0] = volume[...]      # ✅ 基于当日成交量
df['OBV'] = obv.cumsum()                 # ✅ 累计求和

# 使用OBV_diff作为因子
df['OBV_diff'] = df['OBV'].diff()        # ✅ OBV的变化量
```

**结论**: ✅ **无未来函数**
- OBV使用当日成交量和收盘价
- OBV_diff是OBV的变化率
- 累计求和是单调的，无look-ahead

## 二、指标计算中的shift(-1)分析

### 2.1 发现的位置

```bash
src/indicators/wrapper.py:335:  df[f'return_{period}d'] = close.pct_change(period).shift(-period)
src/factor_report.py:86:        df['next_return'] = df['close'].pct_change().shift(-1)
```

### 2.2 用途分析

| 文件 | 用途 | 是否合理 |
|------|------|----------|
| wrapper.py:335 | 计算N日收益率 | ✅ **合理** - 用于信号验证，需要知道持仓期结束时的收益 |
| factor_report.py:86 | 计算下一期收益 | ✅ **合理** - 用于IC分析，验证信号预测能力 |

### 2.3 关键区分

```
未来收益计算 (合理):
  Day T 生成信号 → Day T+1 买入 → Day T+3 卖出
                              ↓
                    用shift(-3)计算3日收益
                              ↓
                    这是信号结果的计算，不是信号生成

指标计算 (避免):
  Day T 计算指标时使用 shift(-1) 引用未来数据 ❌ 这是未来函数
```

**结论**: ✅ shift(-1)只用于计算信号执行后的收益，不参与信号生成

## 三、文献参考：常见技术指标的未来函数风险

### 3.1 学术观点

根据量化交易研究文献：

| 指标 | 未来函数风险 | 说明 |
|------|-------------|------|
| **移动均线 (MA)** | 无 | 只用历史数据 |
| **MACD** | 无 | 基于EMA，滞后指标 |
| **RSI** | 无 | 基于历史价格变化 |
| **布林带** | 无 | 基于历史统计量 |
| **ADX** | 无 | 基于历史TR和DM |
| **KDJ** | 无 | 基于历史随机过程 |
| **SAR** | 无 | 递归计算，滞后 |
| **OBV** | 无 | 基于历史成交量 |

**结论**: 主流技术指标均无未来函数风险

---

### 3.2 需要注意的特殊情况

| 场景 | 风险 | 规避方法 |
|------|------|----------|
| 使用收盘价计算时引用明日收盘价 | ⚠️ 高 | 只用当日收盘价 |
| 使用未对齐的多周期数据 | ⚠️ 中 | 对齐到同一日期 |
| 盘中实时计算信号 | ⚠️ 中 | 收盘后计算 |

**我们的策略**: ✅ 所有信号在收盘后计算，使用当日数据

### 5.2 信号生成流程验证

```
┌─────────────────────────────────────────────────────────────┐
│ Day T 收盘后                                                   │
│ ─────────────────────────────────────────────────────────── │
│ 1. 获取当日数据 (close, high, low, volume)                   │
│ 2. 计算指标:                                                   │
│    - ADX: 使用历史14日TR/DM (✅ 无未来数据)                    │
│    - BB: 使用历史20日均线/标准差 (✅ 无未来数据)               │
│    - SAR: 使用当日价格判断趋势 (✅ 无未来数据)                │
│    - OBV: 使用当日成交量累加 (✅ 无未来数据)                  │
│ 3. 计算加权分数: score = Σ(weight × normalized_factor)       │
│ 4. 判断信号: if score >= threshold: 生成买入信号              │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│ Day T+1                                                    │
│ ─────────────────────────────────────────────────────────── │
│ 5. 执行买入: 按次日开盘价或收盘价买入                          │
│ 6. 持仓N天或触发止损/止盈                                     │
│ 7. 计算收益: return = (exit_price - entry_price) / entry_price│
└─────────────────────────────────────────────────────────────┘
```
