### 二、评估指标体系

| 指标 | 计算方式 | 目标值 | 说明 |

### 三、未来函数检验（必须通过）

```
【未来函数定义】
使用未来数据影响当前决策

【检验方法】
1. 移动平均偏移检验
   - 计算信号发出时的MA值
   - 验证信号不在收盘价之后才满足均线条件

2. 未来数据泄露检验
   - 遍历所有买入信号
   - 检查信号日期后N日内是否"恰好"满足条件
   - N=信号周期+1天内满足条件 → 疑似未来函数

3. 交叉验证
   - 训练期优化的参数
   - 直接应用于测试期
   - 若测试期效果相近 → 无未来函数

【通过标准】
- [ ] 训练期/测试期收益衰减 < 30%
- [ ] 买入信号满足条件不能在信号日期之后才成立
- [ ] 交易记录中无明显"事后诸葛亮"现象
```

---

### 四、过拟合检验（必须通过）

```
【过拟合定义】
策略在训练期过度拟合，无法泛化到测试期

【检验方法】
1. 样本外测试
   - 训练期优化参数
   - 测试期验证效果
   - 测试期收益/训练期收益 > 70% → 无过拟合

2. 参数敏感性分析
   - 绘制 参数 vs 收益 曲线
   - 若曲线平坦 → 参数不敏感 → 不易过拟合
   - 若曲线陡峭 → 参数敏感 → 易过拟合

3. 交叉ETF验证
   - 在ETF A上优化的参数
   - 直接应用于ETF B
   - 若效果相近 → 无过拟合

4. 随机对照实验
   - 随机生成买入信号
   - 对比随机策略 vs 鱼身策略
   - 若鱼身策略显著优于随机 → 非过拟合

【通过标准】
- [ ] 测试期收益 / 训练期收益 > 70%
- [ ] 参数在±10%范围内收益稳定
- [ ] 不同ETF间收益衰减 < 50%
- [ ] 鱼身策略显著优于随机策略(p < 0.05)
```
