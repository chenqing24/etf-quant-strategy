## 实验设计 V3.0

### 一、实验结构

```
实验设计: 多ETF × 多时间段 × 多参数

【ETF选择】(5个)
- 510300 (沪深300)
- 510500 (中证500)
- 159806 (新能源车)
- 512760 (医疗)
- 511010 (纳指)

【时间段】(2个)
- 训练期: 2022-01-01 ~ 2024-06-30
- 测试期: 2024-07-01 ~ 2026-05-28

【参数配置】(6组)
- Cfg1: 止损-5%, 止盈8%, 持仓15天
- Cfg2: 止损-4%, 止盈6%, 持仓10天
- Cfg3: 止损-3%, 止盈5%, 持仓7天
- Cfg4: 止损-6%, 止盈10%, 持仓20天
- Cfg5: 止损-4%, 止盈8%, 持仓12天
- Cfg6: 止损-5%, 止盈6%, 持仓10天

【鱼身因子】(5个条件，4个通过才买入)
- 多头排列: MA5 > MA10 > MA20
- MACD正: MACD柱 > 0
- 鱼身位置: 30% <= 高点距离 <= 85%
- RSI适中: 25 <= RSI <= 65
- 趋势一致: (短期+中期+长期)/3 > 0

总实验数: 5 ETF × 2 时段 × 6 参数 = 60个实验
```

---

### 二、评估指标体系

| 指标 | 计算方式 | 目标值 | 说明 |
|------|----------|--------|------|
| **总收益** | (期末净值-1) × 100% | >30% | 最终收益 |
| **年化收益** | 总收益 / 年数 | >20% | 年化收益 |
| **夏普比率** | (年化收益-无风险)/年化波动 | >0.8 | 风险调整收益 |
| **最大回撤** | max(账户高点-低点)/高点 | <15% | 风险控制 |
| **胜率** | 盈利次数/总次数 | >55% | 交易质量 |
| **盈亏比** | 平均盈利/平均亏损 | >1.0 | 盈亏平衡 |
| **交易次数** | 总交易数 | >10 | 样本量 |
| **训练期IC** | 因子IC均值 | >0.01 | 因子有效 |
| **测试期IC** | 因子IC均值 | >0.005 | 泛化能力 |

---

### 三、评估维度扩展

#### 维度1: 多时间段验证
```
训练期: 2022-01-01 ~ 2024-06-30 (2.5年)
测试期: 2024-07-01 ~ 2026-05-28 (1.8年)

要求:
- 训练期和测试期都要有收益
- 测试期收益不能比训练期差太多(衰减<30%)
```

#### 维度2: 跨ETF稳定性
```
每个参数配置在5个ETF上测试
评估:
- 正收益ETF占比 > 80%
- 收益波动 < 30%
```

#### 维度3: 因子有效性
```
鱼身因子IC验证:
- 多头排列: IC > 0.01
- MACD正: IC > 0.01
- 鱼身位置: IC > 0.005
- 趋势一致: IC > 0.01
```
