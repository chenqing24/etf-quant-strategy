# 行业配置规范

> 规范ETF的行业分类和配置策略

## 1. 概述

行业配置模块管理ETF的行业分类，支持均衡配置和行业轮动两种策略。

## 3. 配置策略

### 3.1 均衡配置（默认）

每只ETF配置相等的资金：
```
单只ETF仓位 = 总资金 / ETF数量
```

### 3.2 行业权重

| 行业 | 权重上限 |
|------|----------|
| 宽基指数 | 40% |
| 科技 | 30% |
| 医疗 | 15% |
| 消费 | 15% |
| 其他 | 20% |

### 3.3 动态调整

| 市场状态 | 调整策略 |
|----------|----------|
| 强势市场 | 超配科技、创业板 |
| 震荡市场 | 均衡配置 |
| 弱势市场 | 超配宽基、防守 |

## 4. 行业过滤

### 4.1 排除规则

以下ETF不纳入投资池：

| 类型 | 示例 | 排除原因 |
|------|------|----------|
| 红利ETF | 512890, 515080 | 强周期 |
| 港股通 | 513550, 159585 | 汇率风险 |
| 证券ETF | 512880 | 强周期 |
| 债券ETF | 511010 | 低收益 |
| 商品ETF | 518880 | 高波动 |

### 4.2 过滤代码

```python
EXCLUDED_PATTERNS = [
    '红利', '港股通', '证券', '债券', '商品',
    '黄金', '原油', '豆粕', '白银'
]
```

## 5. 使用示例

```python
from src.industry_mapping import INDUSTRY_MAPPING, get_industry

# 获取ETF行业
industry, name = get_industry('515050')
print(f"515050 属于 {industry} - {name}")

# 遍历所有ETF
for code, (ind, name) in INDUSTRY_MAPPING.items():
    print(f"{code}: {ind} - {name}")
```
