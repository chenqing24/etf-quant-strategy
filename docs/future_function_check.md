# 未来函数检测报告

## 一、什么是未来函数

**未来函数 (Look-ahead Bias)** 指在回测中使用了回测结束时才能获得的数据，导致回测结果优于实盘表现。

### 典型场景
| 场景 | 错误做法 | 正确做法 |
|------|---------|---------|
| 指标计算 | 使用明日数据 | 只使用历史数据 |
| 交易执行 | 盘中实时下单 | 收盘后计算，次日执行 |
| 止损判断 | 使用收盘价+未来数据 | 只用当前持仓成本 |

│ ┌──────┐        ┌──────┐           ┌──────────┐                 │

│                                                                  │
└─────────────────────────────────────────────────────────────────┘

**关键点**:
- ✅ **信号生成**: 每日收盘后，只使用当日数据
- ✅ **交易执行**: 次日执行，不使用未来价格
- ✅ **退出判断**: 基于入场价格计算，不引用未来数据
