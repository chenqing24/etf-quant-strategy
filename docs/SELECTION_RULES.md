# ETF选股规则文档

## 1. 概述

本文档记录ETF量化策略的选股规则，分为三部分：
1. **排除规则**: 过滤不适合的ETF
2. **初选规则**: 训练期内收益TopN
3. **日常打分**: 7因子模型

---

## 2. 排除规则 (Exclusion Rules)

### 2.1 排除原因分类

| 类别 | 排除原因 | 示例代码 |
|------|----------|----------|
| **港股通** | 港股市场与A股相关性低，流动性差异大 | 159825, 159902, 159915, 159928, 159952 |
| **红利/养老** | 高股息策略与动量策略冲突 | 513360, 515050, 513080 |
| **证券/金融** | 周期性强，波动不适合动量策略 | 512880, 512170, 512200 |

### 2.2 完整排除列表

```python
exclude_codes = {
    # 港股通ETF (5只)
    '159825',  # 港股通50
    '159902',  # 港股通100
    '159915',  # 港股通中小盘
    '159928',  # 港股通创新药
    '159952',  # 港股通消费ETF
    
    # 红利/养老ETF (3只)
    '513360',  # 红利ETF
    '515050',  # 养老产业ETF
    '513080',  # 红利低波动ETF
    
    # 证券/金融ETF (3只)
    '512880',  # 证券ETF
    '512170',  # 券商ETF
    '512200',  # 金融ETF
}
```

### 2.3 排除原因详解

| 类型 | 问题 | 策略考量 |
|------|------|----------|
| **港股ETF** | 汇率风险、交易时段不同、流动性低 | 动量策略在A股更有效 |
| **红利ETF** | 高股息策略与动量策略逻辑冲突 | 红利策略买跌不动量，动量买涨 |
| **证券ETF** | 周期性强、弹性过大 | 牛市中涨太多，熊市中跌太多 |

### 2.4 数据过滤

| 规则 | 阈值 | 原因 |
|------|------|------|
| 数据长度 | ≥ 500天 | 技术指标需要足够历史数据 |
| 排除新上市 | < 500天 | 避免上市不久、数据不足的ETF |

## 3. 初选规则 (Backtest Selection)

### 目标
在历史数据中选出长期表现优秀的ETF，形成股票池。

### 规则

| 参数 | 值 | 说明 |
|------|-----|------|
| 训练期 | 2022-01-01 ~ 2024-12-31 | 3年历史数据 |
| 选股数量 | Top 30 | 按训练期收益率排序 |
| 排除代码 | 见2.2节 | 港股/红利/证券类 |

### 筛选公式

```
训练期收益率 = (期末收盘价 / 期初收盘价) - 1
```

### 流程

```
1. 加载所有ETF数据 (排除<500天的)
2. 排除 exclude_codes 中的ETF
3. 计算2022-2024年收益率
4. 按收益率降序排列
5. 取Top30进入股票池
```

### 输出
选出收益率最高的30只ETF进入日常交易池。

## 4. 日常打分规则 (7因子模型)

### 目标
每日对股票池内ETF打分，选出分数最高的TopN持仓。

### 评分因子

| 因子 | 分数 | 条件 | 逻辑 |
|------|------|------|------|
| **MA120** | +3 | 收盘价 > MA120 | 长期上涨趋势 |
| **MA60向上** | +2 | 最近5日MA60上升 | 中期动量 |
| **MA60** | +2 | 收盘价 > MA60 | 站上中期均线 |
| **MA20** | +1 | 收盘价 > MA20 | 短期趋势 |
| **放量** | +2 | 量比 > 1.5 | 资金流入 |
| **RSI** | +1 | RSI14 < 70 | 未超买 |
| **MACD** | +1 | MACD > 0 | 上涨动能 |

### 完整因子表

| # | 因子 | 分数 | 条件 | 备注 |
|---|------|------|------|------|
| 1 | 站上MA120 | +3 | close > ma120 | 长期趋势确认 |
| 2 | MA60向上 | +2 | ma60当前值 > ma60_5日前 | 中期动量 |
| 3 | 站上MA60 | +2 | close > ma60 | 中期趋势确认 |
| 4 | 站上MA20 | +1 | close > ma20 | 短期趋势 |
| 5 | 放量 | +2 | vol_ratio > 1.5 | 成交量放大 |
| 6 | RSI健康 | +1 | rsi_14 < 70 | 避免追高 |
| 7 | MACD金叉 | +1 | macd > 0 | 动能向上 |

### 触发阈值

| 参数 | 值 | 说明 |
|------|-----|------|
| **总分门槛** | ≥ 6分 | 低于此分不买入 |
| **持仓数量** | Top 2 | 持仓不超过2只 |
| **仓位权重** | [0.6, 0.4] | 第1只60%, 第2只40% |

### 评分流程

```
对股票池内每只ETF:
    1. 获取当日技术指标
    2. 逐项检查7个因子
    3. 累加得分
    4. 记录选股理由

选出得分>=6分的ETF:
    - 按分数降序排列
    - 取Top2持仓
    - 分数相同按成交量排序
```

## 9. 常见问题

**Q: 为什么不买港股ETF?**
> 港股与A股市场相关性不同，汇率风险大，且动量策略在A股历史数据更有效。

**Q: 红利ETF为什么排除?**
> 红利策略是"买跌、高股息"，动量策略是"买涨、追趋势"，两者逻辑冲突。

**Q: 证券ETF为什么排除?**
> 证券板块周期性强，在牛市中涨5-10倍，熊市中跌80%，波动过大不适合动量策略。

**Q: 选股池多久更新一次?**
> 每次回测前会重新根据最新3年数据选一次，确保选入近期表现好的ETF。
