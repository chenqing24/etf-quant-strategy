## 📅 2026-05-25

### ✅ US-005 交易日志增强 + 查询功能 - 已完成

#### 变更内容

**1. 扩展 `TradeRecord` dataclass（`src/trade_tracker.py`）**

新增字段：

| 字段 | 类型 | 说明 |
|------|------|------|
| `realtime_price` | float | 买入时实时价格快照 |
| `price_deviation` | float | 价格偏差率 `(实时-信号)/信号 * 100%` |
| `rsi_14` | float | RSI(14) 值 |
| `day_change_pct` | float | 当日涨跌幅 `%` |
| `score` | int | 策略评分 |

**2. 扩展 `record_buy()` 自动填充实时数据**

- 优先使用 `DataFacade` 热数据层
- 降级使用腾讯API直调 `qt.gtimg.cn`
- RSI(14) 从 cold CSV 计算（`ColdDataManager`）
- `signal_price` 参数用于计算偏差率

**3. 新增 `query_trades(date, code, action)` 查询方法**

```python
# 示例
tracker = TradeTracker('etf_data_live')

# 按日期查询
tracker.query_trades(date='2026-05')

# 按代码查询
tracker.query_trades(code='510300')

# 按行为类型查询
tracker.query_trades(action='buy')

# 组合过滤
tracker.query_trades(date='2026-05-25', code='510300', action='buy')
```

**4. 新增 `export_csv(filepath)` 导出方法**

导出字段：所有 `TradeRecord` 字段（含 US-005 新增字段），编码 `utf-8-sig`（Excel 友好）。

**5. 修改 `src/decision_cli.py`**

新增 CLI 参数：

| 参数 | 模式 | 说明 |
|------|------|------|
| `-m history --date 20260525 --code 510300` | 查询 | 按日期/代码过滤交易记录 |
| `-m export --filepath trades.csv` | 导出 | 导出全量交易记录为CSV |

```bash
# 查询2026年5月所有买入记录
python -m src.decision_cli -m history --date 2026-05 --action buy

# 导出到指定路径
python -m src.decision_cli -m export --filepath ~/Desktop/trades.csv
```

- [x] `query_trades()` 支持按日期/标的查询
- [x] `export_csv()` 能导出CSV文件

### 关于搜索引擎的说明
❌ **搜索引擎不能用于实时数据获取**
- 爬取延迟：分钟到小时级
- 数据非实时：搜索引擎索引的是网页快照
- 股票每秒变化：无法跟踪

✅ **正确方案**：直接爬取金融网站或使用API
