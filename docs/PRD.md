# ETF量化策略项目 - 产品需求文档

## 1. 项目概述

| 项目 | 内容 |
|------|------|
| 项目名称 | ETF量化交易策略系统 |
| 项目类型 | 量化投资回测系统 |
| 核心功能 | 基于技术指标的ETF趋势轮动策略 |
| 目标用户 | 量化投资者 |

---

## 2. 业务背景

### 2.1 问题域

- **问题**：手动选ETF效率低，容易追高被套
- **解决方案**：建立一套基于技术指标的自动化选股系统

### 2.2 量化基础

| 概念 | 定义 |
|------|------|
| 训练期 | 2022-01-01 ~ 2024-12-31（用于选出优质ETF） |
| 测试期 | 2025-05-06 ~ 2026-05-22（用于验证策略有效性） |
| 过拟合 | 用当年数据选ETF再用同年数据测试（应避免） |
| 科学验证 | 用历史数据选出ETF，用未来数据验证 |

## 4. 参数配置

### 4.1 可调参数

| 参数名 | 默认值 | 范围 | 说明 |
|--------|:------:|:----:|------|
| rebalance_days | 5 | 3~30 | 调仓周期（天） |
| score_threshold | 6 | 4~10 | 选股分数门槛 |
| hold_count | 2 | 1~5 | 持仓数量 |
| weights | (0.5, 0.5) | - | 仓位权重 (各50%) |
| stop_loss | -0.10 | - | 止损比例 |
| stop_gain | 0.15 | - | 止盈比例 |
| max_hold_days | 15 | - | 最大持仓天数 |
| market_ma | 60 | 20~120 | 市场过滤均线 |

### 4.2 交易统计（回测结果）

| 指标 | 值 | 说明 |
|------|---:|------|
| 平均持仓天数 | 6.5天 | 实际平均持仓 |
| 最大持仓天数 | 15天 | 达到止损线 |

> 注：因持仓时间短，止盈止损触发频率较低，主要依靠调仓和止损线控制风险。

### 4.2 最优参数（经过验证）

| 参数 | 最优值 | 验证结果 |
|------|:------:|----------|
| **rebalance_days** | **5** | 回撤最小(-22.6%)，夏普最高(2.07) |
| score_threshold | 6 | 与4/5/7/8差异不大 |
| market_ma | 60 | 基准配置，可尝试MA120减少过滤 |

## 8. 问题修复记录

### 8.1 回撤问题（已修复）

| 项目 | 内容 |
|------|------|
| 问题 | MA60市场过滤触发49次，每次清仓导致高买低卖 |
| 现象 | 回撤-55.5%，夏普仅0.33 |
| 根因 | 市场短期波动触发过滤后，立即买入接飞刀 |

**修复方案**：
```python
# 1. 添加冷却期
self.cooldown_days = 0  # 市场转强立即入场

# 2. 追踪市场状态
self.was_bearish_prev = False

# 3. 市场转强时立即入场
market_turned_bullish = self.was_bearish_prev and market_ok
can_reenter = market_turned_bullish or (date_idx - last_trigger) >= cooldown
```

### 8.2 初始买入bug（已修复）

| 项目 | 内容 |
|------|------|
| 问题 | 回测第一天就买入，未检查市场是否上涨 |
| 现象 | 可能买在市场高点 |
| 修复 | 首次买入需等待市场上涨信号 |

**修复代码**:
```python
# 首次买入 - 需等待市场上涨
market_ok = not market_filter or market_filter.is_bullish(first_date)
if market_ok:
    # 执行买入
    ...
```

### 8.3 参数优化路径

| 阶段 | 发现 | 结果 |
|------|------|------|
| 调仓3天 | 收益低(+25.7%) | 继续尝试 |
| 调仓10天 | 收益高但回撤大(-55.5%) | 尝试减少 |
| **调仓5天** | **最佳平衡** | **+51.9%收益,-22.6%回撤** |

### 8.3 MA120测试

MA60过滤触发过于频繁，测试MA120替代：
- MA120: 收益+124.1%, 回撤-53.5%
- 但调仓5天回撤更小，夏普更高

**结论**：当前最优配置为**调仓5天**

### 10.2 配置驱动的策略执行框架

**状态**: 已确认 ✅

**问题**：策略挖掘与执行引擎紧耦合
- 每换一个策略就要改代码
- 结果不可比（执行逻辑可能不同）

**目标**：策略配置与执行引擎完全分离

```
┌──────────────────┐     ┌──────────────────┐     ┌──────────────────┐
│  挖掘系统         │     │   配置输出        │     │   统一引擎        │
│  (Experimenter)  │ ──▶ │   (JSON/DB)      │ ──▶ │   (Executor)     │
│                  │     │                  │     │                  │
│  - IC分析        │     │  factors         │     │  - 逐日循环      │
│  - 权重计算      │     │  weights         │     │  - 持仓管理      │
│  - 参数调优      │     │  config          │     │  - 风控止损      │
└──────────────────┘     └──────────────────┘     └──────────────────┘
                                                          │
                                                          ▼
                                                 ┌──────────────────┐
                                                 │   统一结果        │
                                                 │   BacktestResult  │
                                                 └──────────────────┘
```

**模块说明**：

| 模块 | 文件 | 职责 | 输入 | 输出 |
|------|------|------|------|------|
| 策略配置 | `strategy_config.py` | 配置序列化/反序列化 | Python对象 | JSON/DB |
| 配置存储 | `experiment_store.py` | 配置持久化 | Config对象 | experiments.json |
| 统一引擎 | `universal_executor.py` | 配置驱动执行 | JSON/Config | BacktestResult |

**核心类设计**：

```python
@dataclass
class FactorStrategy:
    """因子策略配置"""
    name: str
    factors: List[str]
    weights: Dict[str, float]           # 因子权重
    direction: Dict[str, str]            # 因子方向 (long/short/neutral)
    score_threshold: float = 0.6         # 分数阈值
    min_factors: int = 2                 # 最少有效因子数


@dataclass
class BacktestConfig:
    """回测配置"""
    stop_loss: float = -0.05             # 止损
    stop_profit: float = 0.10            # 止盈
    hold_days: int = 5                   # 最大持仓天数
    max_positions: int = 2               # 最大持仓数
    commission: float = 0.0003            # 手续费
    slippage: float = 0.001             # 滑点


@dataclass
class ExperimentConfig:
    """实验完整配置"""
    experiment: FactorStrategy
    backtest: BacktestConfig
    data: Dict[str, str]               # 时间范围


class UniversalExecutor:
    """统一执行引擎"""
    
    def load_config(self, config: ExperimentConfig) -> None:
        """加载配置"""
        
    def run(self) -> BacktestResult:
        """执行回测"""
        
    def run_with_store(self, exp_id: int) -> BacktestResult:
        """从存储加载配置并执行"""
```
