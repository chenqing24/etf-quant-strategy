### 1.2 数据流向

```
定时任务 (14:25 工作日)
      │
      ▼
数据预热 (Prefetcher) ──→ 热数据层 (Hot Layer)
      │
      ▼
数据加载 (DataLoader) ──→ 冷数据层 (Cold Layer)
      │
      ▼
指标计算 (Indicator) ──→ 技术指标数据
      │
      ▼
选股评分 (Selector) ──→ Top ETF 列表
      │
      ▼
市场过滤 (MarketFilter) ──→ 买入/卖出信号
      │
      ▼
决策报告 (Reporter) ──→ 交易建议
      │
      ▼
钉钉推送 (DingTalk) ──→ 用户手机
```

## 4. 术语表

### 4.1 核心概念

| 术语 | 英文 | 定义 |
|------|------|------|
| **ETF** | Exchange Traded Fund | 交易型开放式指数基金 |
| **量化策略** | Quantitative Strategy | 基于数学模型的交易策略 |
| **因子** | Factor | 影响资产价格的量化指标 |
| **选股** | Stock Selection | 从候选池中筛选符合条件的标的 |
| **回测** | Backtest | 用历史数据验证策略有效性 |
| **夏普比率** | Sharpe Ratio | 风险调整后的收益指标 |
| **最大回撤** | Max Drawdown | 策略历史最大亏损幅度 |
| **胜率** | Win Rate | 盈利交易占总交易的比例 |

### 4.2 数据相关

| 术语 | 英文 | 定义 |
|------|------|------|
| **热数据** | Hot Data | 今日实时价格，盘中持续更新 |
| **冷数据** | Cold Data | 历史收盘数据，归档后不修改 |
| **MA** | Moving Average | 移动平均线 |
| **RSI** | Relative Strength Index | 相对强弱指数 |
| **MACD** | Moving Average Convergence Divergence | 指数平滑异同移动平均线 |
| **放量** | Volume Surge | 成交量显著放大 |
| **偏离信号** | Price Deviation | 当前价与信号价的偏差 |

### 4.3 交易相关

| 术语 | 英文 | 定义 |
|------|------|------|
| **止损** | Stop Loss | 亏损达到阈值时卖出 |
| **止盈** | Stop Gain | 盈利达到目标时卖出 |
| **移动止盈** | Trailing Stop | 盈利回撤时触发卖出 |
| **调仓** | Rebalance | 调整持仓结构 |
| **持仓天数** | Hold Days | 持有标的时间 |

### 4.4 系统相关

| 术语 | 英文 | 定义 |
|------|------|------|
| **信号价格** | Signal Price | 策略生成的买入参考价 |
| **实时价格** | Realtime Price | 当前市场报价 |
| **场景适配** | Scenario Adapter | 根据使用场景调整输出格式 |
| **cron** | Cron | 定时任务调度器 |

### 4.5 性能指标

| 术语 | 英文 | 定义 |
|------|------|------|
| **收益率** | Return | 投资回报率 |
| **回撤** | Drawdown | 从高点的下跌幅度 |
| **夏普** | Sharpe | 风险调整收益比 = (Rp-Rf)/σp |
| **波动率** | Volatility | 收益的标准差 |
| **盈亏比** | Profit/Loss Ratio | 平均盈利/平均亏损 |

### 4.6 缩写对照

| 缩写 | 全称 |
|------|------|
| MA5 | 5日均线 |
| MA20 | 20日均线 |
| MA60 | 60日均线 |
| MA120 | 120日均线 |
| RSI14 | 14日RSI |
| MACD | 指数平滑异同移动平均线 |
| ETF | 交易型开放式指数基金 |
| CSV | 逗号分隔值文件格式 |
| JSON | JavaScript对象表示法 |
| API | 应用程序编程接口 |
