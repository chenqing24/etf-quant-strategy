# ETF量化决策系统 - 架构设计

## 一、系统架构

```
┌─────────────────────────────────────────────────────────┐
│                     用户层                               │
│  钉钉/PC → QwenPaw Agent → decision_cli                 │
└─────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────┐
│                     入口层                               │
│  decision_cli.py (统一命令行入口)                        │
│  - daily: 每日检查                                       │
│  - eval: 完整评估                                        │
│  - trade: 记录交易                                       │
│  - history: 查看历史                                     │
│  - perf: 绩效分析                                        │
│  - update_pool: ETF池更新                               │
└─────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────┐
│                   数据采集层                             │
│  data_fetcher.py (腾讯API)                               │
│  - 本地缓存 + 增量更新                                   │
│  - 动态ETF池 (etf_pool.json, 25只)                       │
│  - 自动过滤特殊ETF                                       │
└─────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────┐
│                   决策分析层                             │
│  selector.py (7因子选股)                                 │
│  - MA趋势/动量/RSI/MACD/成交量                          │
│  - IC加权因子打分                                        │
│  - 排除规则 (30只特殊ETF)                               │
│                                                     │
│  market_filter.py (市场过滤)                            │
│  - 识别牛熊市                                            │
│  - 只在上涨趋势买入                                      │
└─────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────┐
│                   策略验证层                            │
│  backtest.py (多时段回测)                               │
│  - 2023-2025 / 2024-2026 双时段验证                    │
│  - 计算: 收益/回撤/夏普/胜率                            │
│  - 滚动训练期 → 测试期                                  │
└─────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────┐
│                   报告生成层                            │
│  report_generator.py                                    │
│  - 固定模板报告                                         │
│  - 交易建议放开头和结尾                                 │
│  - ETF名称映射 (25只)                                   │
└─────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────┐
│                   推送层                               │
│  钉钉推送 (通过QwenPaw渠道)                             │
│  - 简化版消息: 操作/标的/价格/止损止盈                   │
│  - 无需webhook配置                                      │
└─────────────────────────────────────────────────────────┘
```

---

## 二、核心模块

| 模块 | 文件 | 职责 |
|------|------|------|
| 数据采集 | `data_fetcher.py` | 腾讯API，本地缓存，增量更新 |
| ETF池管理 | `etf_pool_updater.py` | 股票池维护，每2周更新 |
| 7因子选股 | `selector.py` | 技术指标打分，排除特殊ETF |
| 市场过滤 | `market_filter.py` | 牛熊市识别，趋势判断 |
| 回测引擎 | `backtest.py` | 多时段验证，绩效计算 |
| 交易执行 | `trade.py` | 买入/卖出/止损/止盈 |
| 报告生成 | `report_generator.py` | 固定模板，ETF名称映射 |
| 交易追踪 | `trade_tracker.py` | 持仓记录，历史追踪 |
| 绩效分析 | `performance_analyzer.py` | vs预期/基准对比 |
| 推送通知 | `notifier.py` | 钉钉消息格式化 |
| 统一入口 | `decision_cli.py` | 命令行接口 |

---

## 三、数据流程

```
定时触发 (14:30) / 手动执行
         ↓
    增量更新数据 (腾讯API)
         ↓
    7因子打分 (25只ETF)
         ↓
    筛选Top ETF (≥6分)
         ↓
    多时段回测验证
         ↓
    生成报告 + 钉钉推送
         ↓
    用户执行交易
         ↓
    记录实盘 → 绩效对比
```

## 四、ETF排除规则

| 类别 | 数量 | 原因 |
|------|------|------|
| 港股/境外 | 7只 | 汇率+境外市场影响 |
| 红利ETF | 6只 | 走势与分红基本面相关 |
| 证券/金融 | 5只 | 强周期，波动规律不同 |
| 债券 | 5只 | 与股票走势不同 |
| 商品 | 3只 | 受商品价格主导 |
| 低波策略 | 2只 | 策略驱动，非市场动量 |
| **合计** | **30只** | - |

---

## 五、策略参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 持仓数量 | 1只 | 集中持仓，降低波动 |
| 止损线 | -5% | 快速止损，控制回撤 |
| 止盈线 | +8% | 落袋为安 |
| 移动止盈 | 回撤4% | 盈利超10%后启用 |
| 选股阈值 | 6分 | 7因子得分≥6才推荐 |
| 调仓周期 | 10天 | 到了强制调仓 |
| 最大回撤 | ~21% | 历史验证结果 |
| 夏普比率 | ~0.82 | 风险调整收益 |
